# Agentic Automation Portfolio

Production-style portfolio of three end-to-end agentic automation systems:

1. Autonomous Customer Support & Escalation System
2. Autonomous Document Processing & Compliance Checker
3. AI Sales Pipeline Manager

## Core Stack

- LangGraph for orchestration
- LangChain for tools and integrations
- RAG for knowledge retrieval
- Ollama, vLLM, or TensorRT-LLM compatible runtime configuration
- LangSmith / Langfuse observability hooks
- Streamlit demo interface
- Pytest test coverage
- Docker-ready structure

## Project Goal

This repository demonstrates production-style AI automation systems with:

- multi-agent workflows
- structured outputs
- human approval gates
- exception handling
- config-driven runtime selection
- tests and documentation
- recruiter-friendly demos
